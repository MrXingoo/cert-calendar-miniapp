### 1. 新增页面

- `pages/login/login.js` — 登录成功后跳转日历 Tab

## 部署前必做

### 步骤 1：在云开发控制台创建 `events` 集合

1. 打开[微信开发者工具]
2. 进入「云开发」→「数据库」
3. 点击「创建集合」
4. 集合名称填写：`events`
5. 创建完成后，在「权限设置」中设为「仅创建者可读写」（默认）

### 步骤 2：在云开发控制台创建 `events` 索引（可选，加速查询）

集合 `events` 建议建索引：
| 字段 | 顺序 | 说明 |
|------|------|------|
| `startDate` | 升序 | 按开始日期查询 |
| `endDate` | 升序 | 按结束日期查询 |

## 功能说明

### 月历页面（首页）
- **左右切换月份**（‹ / › 按钮）
- **点击「今」按钮** 回到当月
- **点击日期格子** → 弹出该日事件列表（有事件时）；无事件时直接跳转添加
- **点击事件色条** → 跳转编辑该事件
- **右下角 + 按钮** → 以今天为默认日期添加事件

### 事件编辑页
- **标题**（必填）
- **分类**：工作 / 休假 / 考试 / 个人 / 证照（预置 5 类，可扩展）
- **自定义颜色**：点击分类自动带出颜色，也可点击色块或手动输入色值
- **开始日期**（必填）/ **结束日期**（可选，不填=单日事件，支持跨天）
- **关联证照**（可选）：关联到已有证照
- **备注**（可选）
- **删除**（编辑模式才出现）

### 事件跨天显示
事件设置 `startDate` 和 `endDate` 不同，月历上会在连续的每一天都显示该事件的色条。

## 与证照联动（v2 规划）

当前版本事件是独立于证照的。

## 常见问题

**Q：日历页显示「加载事件失败」**
A：检查云开发控制台是否已创建 `events` 集合，以及集合权限是否正确。

**Q：点击日期没有反应**
A：检查 `app.json` 里 `tabBar.list` 的 `pagePath` 路径是否正确（不要有前导 `/`）。

**Q：事件保存后月历没有立刻显示**
A：`calendar` 页面在 `onShow` 生命周期加载事件，跳转返回后会自动刷新。如未刷新，检查 `wx.navigateBack()` 是否被正确调用。

**Q：农历显示不正确**
A：`utils/lunar.js` 覆盖 1900-2100，如日期超出范围会返回空。可检查控制台是否有相关报错。
