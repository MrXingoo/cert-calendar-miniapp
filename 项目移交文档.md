## 关键功能

### 证照清单
- 证照 CRUD、图片云存储、变更历史快照
- 到期提醒：自定义天数筛选（30/60/90/180/365天）
- 卡片展示：证照缩略图 + 名称 + 到期倒计时 + 时间线
- 点击缩略图可全屏预览大图
- 页面级下拉刷新

### 工作日历
- 6×7 月历网格，含农历、天干地支
- 事件色条按分类颜色显示，跨天连续不断开
- 左右滑动切换月份，顶部箭头点击切换
- 两个分类统计卡：点击选分类、显示该分类本月天数
- 日期点击弹窗查看当日事件，色条点击编辑
- 布局与清单页统一：渐变 header + 头像 + 统计条

### 事件管理
- 全天/跨天事件、分类选择、颜色自动跟随分类
- 关联证照（可选）、备注
- 日期选择器同步显示星期

### 分类管理
- 自定义增删改、20 色调色板 + HEX 自定义色值
- 本地持久化存储、一键恢复默认

### 容错设计
- 登录：本地缓存 → 云函数（3s超时）→ 本地虚拟 ID
- 数据库：所有查询 `.catch()` 返回空，集合不存在不崩溃
- 事件查询：单字段 `startDate` 查询 + 客户端过滤，避免复合索引超时
- 一次性失败保护：事件加载 timeout 后不再重复请求

## 部署步骤

1. 微信开发者工具打开项目根目录，填入 AppID
2. 开通云开发，获取环境 ID
3. 云控制台创建 `events` 集合，建 `startDate` 升序索引
4. 部署 `cloudfunctions/login` 云函数（可选）
5. 编译运行

## 样式架构

`app.wxss` 提供共享顶部组件样式体系，所有页面复用：
- `.gradient-header` — 渐变底色
- `.header-row` — 头像 + 标题行
- `.header-avatar` / `.header-title` / `.header-sub` — 顶部元素
- `.header-action` — 右侧操作按钮
- `.stats-bar` / `.stats-item` / `.stats-num` — 统计条
- `.empty-state` — 空状态占位

页面特有样式写在各自 `.wxss` 中，不相互覆盖。
