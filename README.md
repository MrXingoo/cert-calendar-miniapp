# 证照清单 + 月历 📋

微信小程序 — 个人证照管理 + 日程事件月历

## 功能概览

### 📁 证照清单

| 功能 | 说明 |
|------|------|
| 证照 CRUD | 添加、编辑、删除证照，支持图片上传、到期日期、备注 |
| 到期提醒 | 自定义提醒天数（30/60/90/180/365天），即将到期标记颜色 |
| 搜索筛选 | 按名称搜索，按到期状态筛选 |
| 变更历史 | 每次编辑自动记录变更快照，支持版本回溯 |
| 微信头像 | 微信头像获取与本地持久化 |
| 云存储 | 证照图片存储至微信云存储 |

### 📅 月历

| 功能 | 说明 |
|------|------|
| 月历网格 | 6×7 日历布局，含农历、天干地支、生肖 |
| 事件色条 | 按分类颜色显示事件，跨天事件连续显示（始→中→末圆角衔接） |
| 分类统计 | 顶部展示两个自选分类的本月天数，点击可切换 |
| 事件管理 | 添加/编辑/删除事件，支持全天事件和跨天事件 |
| 分类管理 | 自定义分类：名称、图标（emoji 选择器）、颜色（20色调色板+自定义 HEX） |
| 颜色跟随 | 事件颜色自动跟随分类，无需单独选色 |
| 关联证照 | 事件可关联已有证照，支持取消关联 |
| 日期星期 | 日期选择器同时显示星期，方便选择 |

### 🔐 登录与云开发

| 特性 | 说明 |
|------|------|
| 三级回退登录 | 本地缓存 → 云函数 → 本地虚拟 ID，云函数未部署也能正常运行 |
| 超时保护 | 云函数调用 3 秒超时自动 fallback |
| 容错机制 | 所有数据库查询带 catch，集合不存在时优雅降级 |
| 本地持久化 | 分类配置、用户头像、统计偏好均本地存储 |

## 快速开始

### 环境要求

- 微信开发者工具
- 微信云开发环境（已开通）

### 部署步骤

1. **导入项目**：用微信开发者工具打开本目录，填写 AppID

2. **开通云开发**：点击「云开发」按钮开通，获取环境 ID

3. **创建数据库集合**：

| 集合名 | 说明 | 索引建议 |
|--------|------|----------|
| `certificates` | 证照数据 | — |
| `events` | 事件数据 | `startDate` (升序) |
| `cert_history` | 证照变更记录 | `certId` (升序) |

4. **部署云函数**：在 `cloudfunctions/login/` 目录右键 →「上传并部署」

5. **编译运行**：点击编译即可预览

> 云函数未部署时，登录会自动降级为本地虚拟 ID，不影响功能使用。但要获取真实 openid 以实现多设备同步，建议部署云函数。
